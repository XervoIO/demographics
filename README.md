demographics
============
A project stats API.

## Install

1. Clone
2. `npm install`

## Start

1. `npm start`

If running locally, start mongo or set `MONGO_URL` to an external database.

## Configuration Options

- `HOST`
- `MONGO_URL`
- `PORT`

## API
See [API documentation.](https://github.com/onmodulus/demographics/wiki/API)
